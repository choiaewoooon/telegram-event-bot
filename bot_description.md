# 텔레그램 봇 안내문

## Bot Description (봇 설명)
```
크립토/블록체인 이벤트를 자동으로 분석하고 Notion에 저장하는 AI 봇입니다.

📤 채널 게시물을 포워딩하거나 링크를 보내주세요!
🤖 AI가 자동으로 이벤트 정보를 분석합니다
📊 Notion 데이터베이스에 자동 저장됩니다
```

## About (봇 정보)
```
🤖 이벤트 분석 봇 v3.2

크립토/블록체인 이벤트 정보를 자동으로 수집하고 분석하여 Notion에 저장합니다.

✨ 주요 기능:
• 이벤트 제목/미션 자동 생성
• 시작일/종료일 자동 계산 (통합 날짜 범위)
• 온라인/오프라인 장소 구분
• 총 상금 조건부 표시
• 회차별 상금 상세 분석
• 중복 이벤트 확인

💡 사용법:
1. /start 명령어로 시작
2. 이벤트 게시물을 포워딩하거나 링크 전송
3. AI가 자동으로 분석하여 Notion에 저장
```

## Start Message (시작 메시지) - 이미 코드에 구현됨
```python
"🤖 이벤트 분석 봇 v3.2\n\n"
"📤 채널 게시물을 포워딩하거나 링크를 보내세요!\n"
"🤖 AI가 자동 분석\n"
"📊 Notion에 저장\n\n"
"✨ 주요 기능:\n"
"- 이벤트 제목/미션 자동 생성\n"
"- 시작일/종료일 자동 계산 (통합 날짜 범위)\n"
"- 온라인/오프라인 장소 구분\n"
"- 총 상금 조건부 표시\n"
"- 회차별 상금 상세 분석\n"
"- 중복 이벤트 확인\n"
"- 상금 가치는 수동 입력"
```

## Commands (명령어)
```
start - 봇 시작 및 안내 메시지 보기
```

## Bot Picture 권장 사항
- 크기: 640x360 pixels
- 내용: 이벤트/블록체인 관련 아이콘 또는 로봇 이미지
- 스타일: 심플하고 전문적인 디자인

## 설정 방법

### BotFather에서 설정하기

1. **Bot Description 설정**
```
/setdescription
[봇 선택]
[위의 "Bot Description" 내용 붙여넣기]
```

2. **About 설정**
```
/setabouttext
[봇 선택]
[위의 "About" 내용 붙여넣기]
```

3. **Commands 설정**
```
/setcommands
[봇 선택]
start - 봇 시작 및 안내 메시지 보기
```

4. **Bot Picture 설정** (선택사항)
```
/setuserpic
[봇 선택]
[이미지 업로드]
```
